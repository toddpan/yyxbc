阳阳学编程抖音号视频中的实验源代码  
项目说明：  
arduino/BlinkerIR  
esp8266+红外模块，实现遥控器功能  
功能描述：  
  1、模拟家用电器的遥控器功能。  
  2、接入blinker平台，通过手机可以远程控制。  
  3、接入小爱同学、小度和天猫精灵，可以动动嘴就能控制你的家用电器。  
  4、带红外学习功能，只要用遥控器的电器设备都可以接入。  

硬件准备  
硬件：  
1、红外发射模块 *1  
2、红外接收模块 *1  
3、Nodemcu(826612f) *1  
4、面包板*1  
5、杜邦线*4根
